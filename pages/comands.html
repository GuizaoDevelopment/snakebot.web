👑 Introdução

Snake é um bot perfeito para você que gosta de praticidade, moderação e diversão. Com diversos comandos interativos e sociais para entreter seus membros e engajando seu servidor!

📊 Sobre o Snake

Snake foi desenvolvido em js utilizando a api do discord.js e atualmente hospedado na Square Cloud ficando online 24 horas por dia.

Ainda em versão BETA, com desenvolvimento de funcionalidades e comandos para levar a praticidade até seu servidor! Caso queira me apoiar, prefira pedir para seu servidor grande favorito me adicionar ao invés de me adicionar em um servidor privado. Ou você pode simplesmente votar em mim! Sabia que a cada 10 votos você ganha uma supresa? 💖

🚀 Começando

Para começar a usar o Snake é bem simples! Facilitamos sua vida com a praticidade.

💁🏼‍♂️ Ajuda com o Snake

Ao usar seu primeiro comando, registraremos você em meu banco de dados. Após isso é só você explorar os meus comandos e entrar no mundo da diversão 😎

Precisa de ajuda? Você pode entrar em contato com minha equipe através do meu servidor de suporte.

💖 Adicionando o Snake

> Para me adicionar em seu servidor clique aqui.

Isso requer que você seja o proprietário do servidor ou tenha a permissão "gerenciar servidor".

Após isso, prontinho! Você já pode usar o snake em seu servidor.

🤖 Comandos

Sinônimos:
  • < > : Argumento obrigatório
  • [ ] : Argumento opcional

💰 Economia

  • Daily
    • Descrição : Receba sua quantia diária de snakecoins.
    Uso : /daily
    Nota: Você só pode pegar o daily após 24 horas. Esse comando é essencial lembrar de usar diariamente caso você queira utilizar suas snakecoins.

  • Saldo (atm)
    • Descrição : Veja quantos snakecoins, você, ou outros usuários possuem.
    Uso : /snakecoin atm [ user ]

    Exemplo:
  • Pagar
    • Descrição : Transfira uma quantia de Snekcoins para algum usário ou amigo.
    Uso : /snakecoin pay < user > < quantia >

    Exemplo:
  • Top
    • Descrição : Veja o top global dos mais ricos de snakecoins.
    Uso : /snakecoin top [ page ]

    Exemplo:
  • Rinha de emoji
    • Descrição : Crie uma rinha e desafie seus amigos para jogar com você!.
    Uso : /bet emoji rinha < quantia > [ limite ]

  • Coinflip
    • Comandos de apostas envolvendo o famoso "Cara ou coroa" 💸.
      • coinflip bot
        • Descrição : Cansado de esperar amigos para fazer suas apostas? Tente sua sorte jogando com um bot.
        Uso : /bet coinflip bot < quantia >
        Nota: Você precisa ser um usuário premium para utilizar o comando
        Exemplo:
      • coinflip friends
        • Descrição : Aposte suas snakecoins com seus amigos.
        Uso : /bet coinflip friends < user > < quantia >

        Exemplo:
      • coinflip status
        • Descrição : Veja o status das apostas, seu, ou de outros usuários.
        Uso : /bet coinflip status [user]

        Exemplo:

    📑 Informações

    • Ajuda
      • Descrição : Veja algumas informações necessárias para te ajudar!
      Uso : /help
    • Vote
      • Descrição : Vote em mim no top gg e me ajude a crescer!
      Uso : /vote
      Nota: Sabia que a cada 10 votos você ganha uma recompensa especial?

    • User
      • Comandos de informações de usuário/membros.
        • user info
          • Descrição : Veja as informações suas, ou de outros usuário.
          Uso : /user info [ user ]
          Nota: Comando ainda em fase BETA!
        • user banner
          • Descrição : Veja o banner atual de algum usuário.
          Uso : /user banner [ user ]
        • user avatar
          • Descrição : Veja o avatar atual de algum usuário.
          Uso : /user avatar [ user ]
    • Servidor
      • Comandos de informações do servidor.
        • server memebrCount
          • Descrição : Veja a quantidade atual de membros no servidor.
          Uso : /server memebrcount
        • server info
          • Descrição : Veja as informações do servidor.
          Uso : /server info
        • server banner
          • Descrição : Veja o banner atual do servidor.
          Uso : /server banner
        • server ícone
          • Descrição : Veja o ícone atual do servidor.
          Uso : /server icon

    🔐 Moderação

    • Sistema de entrada
      • Descrição : Configure um canal e mensagens para definir quando um membro novo entrar
      Uso : /config welcome < canal >
    • Autrole
      • Descrição : Defina um cargo para um membro receber assim que entrar no servidor.
      Uso : /config autorole [ cargo ]
